TESTING EMAIL SYSTEM
---------------------
For development/testing, change EMAIL_BACKEND in settings.py to:

1. Console backend (prints to console):
   django.core.mail.backends.console.EmailBackend

2. File backend (saves to sent_emails/ folder):
   django.core.mail.backends.filebased.EmailBackend

3. Memory backend (stores in memory for testing):
   django.core.mail.backends.locmem.EmailBackend

Then switch back to smtp.EmailBackend for production.

TROUBLESHOOTING
---------------
Email not sent but no error:
- Check console/logs for exception messages
- Verify EMAIL_BACKEND is set to SMTP

Gmail authentication failed:
- Use App Password (not regular password)
- First enable 2-Factor Authentication on Gmail
- Generate app password at https://myaccount.google.com/apppasswords

Other email providers:
- Gmail: Use SMTP settings above
- Yahoo: smtp.mail.yahoo.com, port 587, TLS enabled
- Outlook: smtp-mail.outlook.com, port 587, TLS enabled
- Custom: Check your provider's SMTP documentation

Test email sending manually:
 python manage.py shell
 >>> from django.core.mail import send_mail
 >>> from django.conf import settings
 >>> result = send_mail(
     'Test',
     'This is a test email',
     settings.DEFAULT_FROM_EMAIL,
     ['recipient@example.com']
 )
 >>> print(result)  # 1 = success, 0 = failed
